# binance-proxy
binance proxy
local `candle` and `orderbook` and `exchangeinfo`

not support `24hour`.Please use orderbook instead

```
      -f string
            futures bind address. (default ":8091")
      -fakekline
            enable fake kline.
      -s string
            spot bind address. (default ":8090")
      -v    print debug log.
```
